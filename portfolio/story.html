Story Page Development

  • Client: Pete (freelance)
  • Date: February 2015

Development of a story webpage for a client. Client had a clear idea of what he wanted the page(s) to look like and how it should function. Site was built with HTML, CSS, and minimal, pure JavaScript from specs and assets delivered by client. The webstory has some parallax elements where certain images stay in place while other elements scroll over the image as user scrolls down the page. Users also have the ability to control photo changes with scrolling, resulting in something like a gif that the user controls the speed of.
